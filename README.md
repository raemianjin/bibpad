# 성경·찬송 (BibleApp)

성경을 읽고, 찬송가·복음성가 가사를 직접 등록해 보관할 수 있는 안드로이드 앱입니다.

## 기능

- **성경 읽기** — KJV(King James Version) 66권 전체, 1,189장 수록
  - 구약/신약 책 목록, 책 이름 검색
  - 장 단위 가로 스크롤 이동, 이전/다음 장 버튼
  - 글자 크기 6단계 조절
  - 구절 탭 → 클립보드 복사
  - 구절 길게 누르기 → 북마크 토글
- **찬송가/복음성가** — 사용자가 제목과 가사를 직접 입력해 등록
  - 목록 검색, 편집, 삭제
- **북마크** — 저장한 구절 모아 보기, 탭하면 해당 위치로 이동

## v2.1.0 — 패드(태블릿) 지원 (단일 APK)

폰과 패드를 **하나의 APK**로 동시에 지원합니다. 기기의 `smallestScreenWidthDp`
(600dp 이상)를 런타임에 감지해 자동으로 레이아웃을 전환하므로, 폰에서는 기존
동작이 그대로 유지되고 패드에서는 넓은 화면에 맞춰 다음이 적용됩니다.

- **성경 본문** — 패드에서는 한 줄 길이를 읽기 좋은 폭(약 720dp)으로 제한하고
  가운데 정렬해, 가로로 늘어져 읽기 힘든 문제를 없앴습니다.
- **책 목록** — 패드에서 2열(와이드 화면은 3열) 그리드로 표시해 66권을 한눈에.
- **메모 목록** — 패드에서 2~3열 카드 그리드로 표시.

### 메모 장표 — 디자인 템플릿 6종 + 폰/패드 공유

메모 뷰어를 한 장표(이미지 카드)로 보여주는 기능을 다양화했습니다.

- 상단 칩에서 장표 디자인을 즉시 전환: **클래식 / 종이 / 미니멀 / 나이트 / 하늘 / 구절강조**
  (마지막 선택은 자동 저장됩니다)
- **공유 시 형식 선택** — "공유"를 누르면 **폰용(세로 1080px)** / **패드용(가로 1600px)**
  을 고를 수 있고, 선택한 폭으로 고정 렌더링하여 어느 기기에서도 동일한 결과물이
  만들어집니다. 패드용은 사진을 2열 그리드로 배치합니다.
- 공유 이미지는 기기 화면을 그대로 캡처하지 않고 화면에 붙지 않은 별도 카드를
  목표 픽셀 폭으로 그려 PNG로 내보내므로 글자·여백 비율이 일정합니다.


## 빌드 방법

1. Android Studio (Hedgehog 이상 권장) 설치
2. 이 폴더를 Android Studio에서 `Open` 으로 엽니다
3. Gradle 동기화가 끝나면 상단 ▶(Run) 버튼으로 실행하거나,
   터미널에서 다음 명령으로 APK를 생성합니다:

   ```
   ./gradlew assembleDebug      # macOS / Linux
   gradlew.bat assembleDebug    # Windows
   ```

4. 생성된 APK 위치:
   `app/build/outputs/apk/debug/app-debug.apk`

   기기에 설치하려면:
   ```
   ./gradlew installDebug
   ```
   또는 `app-debug.apk` 파일을 안드로이드 기기로 옮겨 직접 설치합니다.
   (기기 설정에서 "출처를 알 수 없는 앱 설치" 허용 필요)

## 환경

- minSdk 24 (Android 7.0) / targetSdk 34
- Kotlin, AndroidX, Material 3
- 외부 서버 통신 없음 — 모든 데이터는 기기 내부에만 저장

## 저작권 안내

- **성경 본문**: King James Version. 퍼블릭 도메인이라 자유롭게 배포·사용 가능합니다.
- **한국어 성경 번역본**(개역개정·새번역 등)은 대한성서공회 등이 저작권을
  보유하고 있어 이 앱에 포함하지 않았습니다.
- **찬송가/복음성가 가사**는 앱에 기본 수록하지 않습니다. 사용자가 직접
  입력하며, 입력하는 가사에 대한 저작권 책임은 사용자에게 있습니다.
  21세기 찬송가 등 저작권이 있는 가사를 무단으로 배포하지 마세요.

이 앱을 마켓에 배포하려면 `applicationId`, 앱 이름, 아이콘을
본인 것으로 바꾸고, 서명 키로 release 빌드를 하면 됩니다.
